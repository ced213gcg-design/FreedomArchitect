# Daily Execution Standard

## Minimum Daily Output
- Applications: 5
- Recruiter messages: 5
- Follow-ups: 3

## Morning Sequence
1. Mission State
2. Executive Board
3. Job Finder
4. Top target selection
5. Application generation
6. Audit
7. Outreach
8. Logging

## Evening Sequence
1. Response review
2. Follow-up queue
3. Metrics update
4. Weakness notes
5. Next-day priority
