# Approval Matrix

## Auto Execute
Allowed when:
- confidence >= 90
- no public-facing risk
- no duplication detected
- no unresolved audit issue

## Review Required
Required when:
- confidence between 70 and 89
- public-facing output
- recruiter-facing output
- strategic message or proposal
- visible brand impact

## Block
Required when:
- confidence < 70
- broken links or naming
- duplicate phrasing too close to recent output
- unclear role alignment
- contradictory or low-quality text
