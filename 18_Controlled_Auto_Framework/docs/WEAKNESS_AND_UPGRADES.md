# Weaknesses and Missing Upgrades

## Current Weak Point
The system is better at construction than conversion.
Primary gap: applications + follow-ups + conversations must rise.

## Critical Missing Upgrades
1. Secrets management
   - Never store sensitive credentials in scripts or repos.

2. Backup and rollback
   - Snapshot key config and docs before large changes.

3. Duplicate suppression
   - Compare new outreach/application text against last 10 outputs.

4. Rate limiting
   - Prevent outreach bursts that look spammy.

5. Observability
   - Every role should emit simple status: started, completed, blocked.

6. Approval inbox
   - One place where REVIEW alerts queue for your approval.

7. Mobile publish path
   - Stable browser-accessible dashboard for phone use.

8. Source quality monitoring
   - Penalize weak job sources over time.

9. Response tracking
   - Track replies, ghosting, and follow-up conversion.

10. Public/private separation
   - Keep proof public, keep operations and sensitive data private.
