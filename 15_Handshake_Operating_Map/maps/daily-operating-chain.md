# Daily Operating Chain

## Morning
1. Check Mission State Dashboard
2. Check Executive Summary Board
3. Run Job Finder
4. Score and select targets

## Midday
5. Generate application packages
6. Submit aligned applications
7. Log application actions

## Afternoon
8. Generate recruiter outreach
9. Send recruiter messages
10. Log outreach

## Evening
11. Review GitHub / LinkedIn alignment
12. Update daily command dashboard
13. End-of-day review

## Daily Minimum Standard
- 5 applications
- 3 recruiter messages
- 2 follow-ups

## Daily Rule
No day ends without proof of execution.
