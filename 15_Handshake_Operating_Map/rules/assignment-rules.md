# Assignment Rules

## Commander
- approves final actions
- chooses strategic direction
- executes submissions

## ChatGPT
- builds systems
- writes aligned materials
- coordinates workflow
- identifies structural gaps

## GitHub
- stores projects
- stores README systems
- displays evidence

## LinkedIn
- carries headline, about, featured links
- hosts visibility and outreach

## FreedomArchitect
- tracks status
- logs actions
- controls dashboards
- stores command systems

## Job Finder
- finds and ranks targets

## Application Generator
- creates application package

## Recruiter Outreach Engine
- creates recruiter message and follow-up

## Audit Branch
- blocks weak output
- confirms release readiness
