# Communication Rules

## Rule 1
Every tool must have one clearly defined primary role.

## Rule 2
No public release occurs without audit.

## Rule 3
No application is sent without logging.

## Rule 4
No recruiter message is sent without follow-up readiness.

## Rule 5
No job lead is trusted without scoring.

## Rule 6
GitHub holds proof. LinkedIn amplifies proof. FreedomArchitect tracks proof.

## Rule 7
ChatGPT defines and coordinates. Commander approves and executes.

## Rule 8
Every repeated error becomes a refinement, not a loop.

## Rule 9
If a system is unclear, mission state becomes unstable.

## Rule 10
Execution must always move toward interviews, remote opportunity, and stronger operational proof.
