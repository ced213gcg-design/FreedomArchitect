# Daily Command Dashboard

## Purpose
This dashboard acts as the daily operational command center for execution, visibility, and control.

## Core Components
- today-command.md
- status/mission-status.md
- status/kpi-tracker.md
- briefings/daily-briefing-template.md
- reviews/end-of-day-review.md
- scripts/dashboard.sh

## Daily Use Sequence
1. Open dashboard
2. Review mission status
3. Review KPI tracker
4. Define today's command priorities
5. Execute
6. Log results
7. End-of-day review

## Rule
No day begins without command clarity.
No day ends without review.
