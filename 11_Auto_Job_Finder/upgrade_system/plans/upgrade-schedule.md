# Upgrade Schedule

## Daily (5–10 min)
- Run job finder
- Review top 10 results
- Log source performance

## Every 3 Days
- Remove 1 weak source
- Add 1 stronger source
- Adjust keyword scoring

## Weekly (30–60 min)
- Refine criteria.json
- Improve keyword targeting
- Review missed opportunities
- Strengthen scoring weights

## Monthly
- Replace entire source list if needed
- Add new platforms (company sites, niche boards)
- Upgrade extraction logic
