# Application Engine

## Purpose
This system controls job applications, follow-ups, recruiter outreach, scoring, and next-step execution.

## Core Components
- logs/daily-application-log.md
- pipeline/application-pipeline.md
- pipeline/recruiter-outreach.md
- scorecards/role-scorecard-template.md
- templates/follow-up-template.md
- templates/recruiter-message-template.md
- next-actions.md

## Workflow
1. Find role
2. Score role
3. Apply
4. Log application
5. Send recruiter outreach
6. Track follow-up date
7. Update pipeline status
8. Archive outcome

## Rule
No application is sent without tracking.
No follow-up is missed.
No opportunity disappears without intelligence capture.
