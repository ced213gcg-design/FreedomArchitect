# SOC Analyst Lab

## Overview
This project simulates a Security Operations Center (SOC) environment.

The goal is to detect, analyze, and respond to suspicious activity using real-world cybersecurity workflows.

---

## Scenario
Simulated alert triggered from abnormal network traffic.

Initial indicators suggested:
- Possible unauthorized access attempt
- Suspicious IP communication
- Unusual traffic patterns

---

## Investigation Steps
1. Reviewed network logs
2. Identified source and destination IPs
3. Analyzed traffic behavior
4. Correlated activity with known attack patterns

---

## Findings
- Suspicious IP identified
- Traffic pattern consistent with scanning behavior
- No confirmed breach, but high-risk activity detected

---

## Actions Taken
- Flagged IP for monitoring
- Recommended firewall rule update
- Logged incident for further review

---

## Tools Used
- Wireshark
- Nmap
- Linux CLI

---

## Conclusion
This lab demonstrates the ability to:
- Analyze network traffic
- Investigate alerts
- Document findings clearly
- Think like a SOC analyst
