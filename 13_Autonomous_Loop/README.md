# Fully Autonomous Loop

## Purpose
Runs the local opportunity pipeline from jobs -> selection -> application package -> logging -> dashboard updates.

## Sequence
1. Run job finder
2. Read ranked jobs
3. Select a job
4. Generate cover letter + resume bullets
5. Log application event
6. Update mission-state and executive board
7. Push updated system state

## Note
This loop prepares and tracks execution. It does not auto-submit on external job sites.
